# Keyword Usage Rules

- Use the primary keyword in the H1 when natural.
- Use the primary keyword in the meta title.
- Use the primary keyword in the first paragraph.
- Use secondary keywords naturally across H2 and body content.
- Do not overuse keywords.
- Write for humans first and search engines second.
- Each page should target one main search intent.
- Avoid duplicate meta titles.
- Add internal links from every product page to related products and contact page.
