# Existing Brand Style Requirement

The new website should keep a similar visual direction to the existing Cooper Ninve website.

The goal is not to create a completely different brand identity, but to modernize and improve the existing website for:
- SEO
- Paid campaigns
- Better lead generation
- Better mobile experience
- Clearer user journeys
- Stronger trust presentation

## Design Direction Based on Existing Website

Use the existing website as visual reference for:
- General brand feeling
- Corporate insurance tone
- Conservative professional look
- Color direction
- Logo placement
- Trust-based layout
- Business / insurance atmosphere

The new design should feel like an upgraded version of the current Cooper Ninve website, not a completely new unrelated brand.

Avoid:
- Too much animation
- Startup-style flashy gradients
- Overly casual design
- Bright aggressive colors
- Cartoon illustrations
- Design that feels unrelated to the current brand

Prefer:
- Clean corporate layout
- White and navy backgrounds
- Professional cards
- Strong but conservative typography
- Clear sections
- Trust badges
- Partner logo strip
- Insurance / business visual style

# Logo & Partner Logos

## Cooper Ninve Logo

The website must include the official Cooper Ninve logo.

Logo placement:
- Header, right side in RTL layout
- Footer
- Optional hero trust area
- Favicon / browser icon if available

Logo requirements:
- Upload logo in SVG or PNG format
- Use transparent background if possible
- Keep correct proportions
- Do not stretch or distort the logo
- Make sure logo is readable on both light and dark backgrounds

Suggested asset folder:

`/assets/logo/`

Suggested file names:
- `cooper-ninve-logo.svg`
- `cooper-ninve-logo.png`
- `cooper-ninve-logo-white.svg`
- `favicon.ico`

## Partner / Market Logos

The website should include a partner / market logos section, only using logos that Cooper Ninve is allowed to display publicly.

Possible section title:
השותפים והשווקים שלנו

Alternative titles:
- שווקים ושותפים בינלאומיים
- פתרונות באמצעות שווקים בינלאומיים
- גישה לשווקים וחתמים מובילים

Important:
Do not imply that every logo is a direct insurer of every product.
Use careful wording.

Recommended wording:
"קופר נינוה פועלת מול שווקים, חתמים וספקי ביטוח בינלאומיים לצורך התאמת פתרונות ביטוח לסיכונים מקצועיים ומסחריים."

Avoid wording:
- “All products are insured by these partners”
- “Guaranteed Lloyd’s coverage”
- “Official partner of every logo shown”
- Any claim that is not legally approved

Suggested asset folder:

`/assets/logos/partners/`

Suggested file names:
- `lloyds-logo.svg`
- `tokio-marine-kiln-logo.svg`
- `qbe-logo.svg`
- `beazley-logo.svg`
- `cfc-logo.svg`
- `other-partner-logo.svg`

# Asset Quality Requirements

All logos should be:
- High resolution
- Transparent background if possible
- SVG preferred
- PNG acceptable
- Consistent height in display
- Optimized for web
- Not pixelated
- Not stretched

Recommended display height:
- Header logo: 48–70px depending on layout
- Partner logos: 36–60px height
- Footer logo: 60–90px
