# Meta Titles & Descriptions - Cooper Ninve

## Goal
Define SEO-friendly meta titles and descriptions for all core pages, product pages, landing pages, and future content pages.

Rules:
- Meta title should usually be up to 55–60 characters.
- Meta description should usually be up to 150–160 characters.
- Each page should have a unique title and description.
- Primary keyword should appear naturally.
- Hebrew should be clear, professional, and conversion-focused.
