# Assets Guide - Cooper Ninve Website

## Goal
This file explains how CODAX should use the uploaded brand assets, partner logos, and current website screenshots.

---

# Official Logo

Use the official Cooper Ninve logo from:

/assets/logo/

Possible files:
- cooper-ninve-logo.svg
- cooper-ninve-logo.png
- cooper-ninve-logo-white.svg
- cooper-ninve-logo-white.png
- favicon.ico

Logo placement:
- Header, right side in RTL layout
- Footer
- Favicon
- Optional hero trust area

Logo rules:
- Do not stretch the logo
- Do not distort proportions
- Use transparent background when available
- Use the white logo version only on dark backgrounds
- Keep the logo visually similar to the existing website

---

# Partner Logos

Use approved partner / market logos from:

/assets/logos/partners/

The partner logo section should appear on:
- Homepage
- About page
- Insurance Agents page

Recommended section title:
השותפים והשווקים שלנו

Recommended wording:
קופר נינוה פועלת מול שווקים, חתמים וספקי ביטוח בינלאומיים במטרה לספק פתרונות ביטוח מתקדמים לסוכני ביטוח, עסקים וסיכונים מורכבים.

Important:
Do not imply that every partner logo applies to every insurance product.
Do not imply guaranteed coverage.
Do not use any third-party logo unless Cooper Ninve is allowed to display it publicly.

If logo approval is unclear, use placeholder containers and mark:
Partner logo pending approval.

# Existing Website Visual Reference

Use screenshots from:

/assets/reference-current-site/

And also review the live website:

https://cooper-ninve.com/

The new website should feel like an upgraded version of the current Cooper Ninve website.

Preserve:
- Corporate insurance atmosphere
- Conservative professional tone
- Similar brand feeling
- Similar logo prominence
- Similar navy / white / business color direction

Improve:
- SEO structure
- Mobile experience
- Lead generation
- CTA visibility
- Page hierarchy
- Product page depth
- Landing pages for paid campaigns

Do not create a completely new unrelated brand identity.
Do not make the website look like a flashy startup.
Do not use cartoon illustrations or aggressive gradients.
