# Brand Assets & Existing Website Reference

Use the existing Cooper Ninve website as the main visual reference:

https://cooper-ninve.com/

Also use the uploaded screenshots inside:

/assets/reference-current-site/

Use official Cooper Ninve logo files from:

/assets/logo/

Use approved partner / market logos from:

/assets/logos/partners/

The new website should feel like a modernized and improved version of the existing Cooper Ninve website.

Keep:
- Similar corporate insurance tone
- Similar brand feeling
- Similar color direction
- Professional and conservative design
- Strong trust-based presentation

Improve:
- SEO
- Paid campaign landing pages
- Mobile experience
- Lead generation
- Clearer navigation
- Product-specific pages

Do not create a completely different visual identity.
Do not use flashy startup design.
Do not use placeholder logos in production.
